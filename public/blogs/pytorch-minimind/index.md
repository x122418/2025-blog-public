<details>
	<summary>torch.arange</summary>
  	用于创建一个一维张量（Tensor），其值是一个等差数列。

```python
	# 从0开始 到10结束（不包含） 步长为2
	torch.arange(0, 10, 2)
	# tensor([0, 2, 4, 6, 8])
	
	# 步长可以为浮点数
	torch.arange(0, 1, 0.25)
	# tensor([0, 0.25, 0.5, 0.75])

	# 步长可以为负数
	torch.arange(5, 0, -1)
	# tensor([5, 4, 3, 2, 1])
```
